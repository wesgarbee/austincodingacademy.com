Learn to Code Workshops

with Austin Coding Academy

Sign up for one of our free workshops and learn the basics of coding. Whether you’re considering a career in web development, looking to understand a bit of code to launch your own website, or you just want to learn something new – our Learn to Code Workshops are a great way to get introduced to the world of web development. In this 2-hour workshop we’ll cover:

  • Coding in HTML and CSS
  • Using a code editor (VScode) and code repository (Git)
  • Deploying your website using Github
  • Understanding how to add JavaScript to your website

After finishing the workshop, you’ll leave with your own live website and an understanding of the basics of web development. You’ll also be more than ready to join one of Austin Coding Academy’s Full Stack courses if you have your sights set on becoming a professional web developer!
